# 🎮 Volvismo 3D – Quiz Interativo

**Um jogo educativo em 3D feito especialmente para o GitHub!**

Baseado no texto que você forneceu sobre o **Volvismo** — o modelo de produção industrial revolucionário da Volvo na Suécia (décadas de 1960-1970).

Você controla um trabalhador autônomo dentro de uma fábrica sueca dos anos 70 e responde perguntas sobre o Volvismo coletando orbes de conhecimento.  
Tudo em **terceira pessoa**, com controles WASD, ambiente industrial bonito e totalmente inspirado no conceito de valorização do trabalhador.

---

## ✨ O que é o Volvismo?
O Volvismo substituiu a linha de montagem rígida por **grupos autônomos**, com treinamento contínuo, alta qualificação, ergonomia e automação flexível — invertendo a lógica de que a máquina dita o ritmo de trabalho.

Este jogo transforma esse conceito em uma experiência divertida e educativa!

---

## 🎯 Como Jogar

1. Clone o repositório ou baixe o arquivo `index.html`
2. Abra o arquivo **`index.html`** diretamente no navegador (não precisa de servidor!)
3. Use as teclas:
   - **WASD** ou **Setas** → mover o trabalhador
   - Aproxime-se dos **orbes laranja brilhantes** para responder as perguntas
4. Responda as 6 perguntas baseadas no texto original
5. Quanto mais acertos, maior sua pontuação!

**Objetivo:** Transformar a fábrica tradicional em uma fábrica de grupos autônomos!

## 🚀 Como Publicar no GitHub Pages (em 2 minutos)

1. Crie um repositório novo no GitHub
2. Faça upload dos dois arquivos (`index.html` + `README.md`)
3. Vá em **Settings → Pages**
4. Em “Source”, escolha **Deploy from a branch** → `main` → **Save**
5. Espere 30 segundos e acesse:  
   `https://SEUUSUARIO.github.io/volvismo-3d-quiz`

Pronto! Seu jogo educativo estará online para todo mundo jogar.

## 📝 Perguntas do Jogo
Todas as 6 perguntas e respostas foram retiradas **exatamente** do texto que você forneceu:

- O que é o Volvismo?
- Em qual período e país foi desenvolvido?
- Qual é o foco principal?
- O que substitui a linha de montagem rígida?
- Principais características
- Qual lógica o Volvismo inverte?
